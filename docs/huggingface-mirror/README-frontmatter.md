# README front matter for the Hugging Face mirror

Put this YAML block at the very top of the README.md in the
OrestisIqtaxi/accesseye-gemma3n-e2b model repo:

```yaml
---
license: gemma
license_name: gemma-terms-of-use
license_link: https://ai.google.dev/gemma/terms
---
```

Below the front matter, keep a sentence like this in the README body:

> Use of this model is subject to the use restrictions in Section 3.2 of the
> Gemma Terms of Use (https://ai.google.dev/gemma/terms) and the Gemma
> Prohibited Use Policy (https://ai.google.dev/gemma/prohibited_use_policy).

Also upload the `NOTICE` and `GEMMA_TERMS.md` files from this folder to the
root of the model repo.
